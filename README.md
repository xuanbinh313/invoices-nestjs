# Take Note
## New Project
```sh
nest new ecommerce-app
```
## Add packages need
1. Validation
```sh
yarn add class-validator class-transformer
```
2. jwt and hashing password
```sh
yarn add @nestjs/jwt 
yarn add bcrypt
yarn add -D @types/bcrypt
```
3. database lib (typeorm)
```sh
yarn add @nestjs/typeorm typeorm sqlite3
```
